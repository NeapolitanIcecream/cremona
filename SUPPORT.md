# Support

Use GitHub Issues for support in this repository.

## Open an issue when you need to

- report a bug in the CLI, report generation, packaging, or docs
- ask a usage question about installation, CI setup, profiles, or baseline handling
- propose a feature or integration

## Before you open an issue

- read [README.md](README.md)
- read [docs/README.md](docs/README.md)
- check [docs/troubleshooting.md](docs/troubleshooting.md)

## Use the right channel

- Bug reports and feature requests: GitHub Issues
- Security problems: [SECURITY.md](SECURITY.md)

This project does not use GitHub Discussions at this stage.
