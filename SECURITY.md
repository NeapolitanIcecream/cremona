# Security Policy

## Supported versions

| Version | Supported |
| --- | --- |
| `0.1.x` | Yes |

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability.

Use GitHub private vulnerability reporting for this repository when it is available. If private reporting is not enabled yet, contact the maintainer through GitHub and include:

- the affected version
- the impact
- a minimal reproduction
- any known workaround

You can expect an initial response within 5 business days. Please give the maintainer time to investigate and ship a fix before discussing the issue in public.
